A Bioinformatics project developed in **C** that detects **Single Nucleotide Polymorphisms (SNPs)** by comparing DNA sequences and identifying nucleotide differences.

---

## 📖 Overview

Single Nucleotide Polymorphisms (SNPs) are variations at a single nucleotide position in DNA sequences. This project compares DNA sequences and identifies mismatched nucleotides that represent SNPs.

This application demonstrates the use of **C programming** in Bioinformatics for DNA sequence analysis.

---

## ✨ Features

- ✅ Reads DNA sequences
- ✅ Validates nucleotide sequences
- ✅ Converts lowercase letters to uppercase
- ✅ Detects SNP positions
- ✅ Reports mismatched nucleotides
- ✅ Calculates sequence similarity
- ✅ Displays mutation information
